## 套件
`pip install selenium` 載入selenium套件\
`https://chromedriver.chromium.org/downloads` 下載driver套件
## 語法
`webdriver.Chrome`開啟chrome\
`driver.get` 取得網址\
`find_elements` 尋找網頁內容位置
